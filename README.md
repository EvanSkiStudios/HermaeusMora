# HermaeusMora
An AI chatbot designed to seek, use, and retain information from online sources or conversations   
[<img src="https://images.uesp.net/thumb/4/42/SR-npc-Hermaeus_Mora.jpg/600px-SR-npc-Hermaeus_Mora.jpg" width="250"/>](image.png)  


[Hermaeus Mora](https://en.uesp.net/wiki/Lore:Hermaeus_Mora) is themed after the Daedric Prince of Forbidden Knowledge from the Elder Scrolls Series  
This project is available under the [MIT License](LICENSE.txt). 

## Elements:  
Ollama AI for chatting  
Basic HTTP requests for web pulling  
Docling for the conversion of documents and tokenizing
